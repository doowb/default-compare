# default-compare [![NPM version](https://badge.fury.io/js/default-compare.svg)](https://npmjs.org/package/default-compare) [![Build Status](https://travis-ci.org/doowb/default-compare.svg?branch=master)](https://travis-ci.org/doowb/default-compare)

> Basic sort algorithm that has similar behavior to Array.prototype.sort for null and undefined, but also allows sorting by an object property.

## Installation

```sh
$ npm install --save default-compare
```

## Usage

```js
var defaultCompare = require('default-compare');
defaultCompare();
```
